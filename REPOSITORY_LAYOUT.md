# Struttura repository

La repository e divisa in due parti:

## Release installabile

`CrimsonDesert_ItalianVoiceMod_GITHUB_READY_v0.2_20260524/`

Contiene la mod installabile, i file WEM italiani, installer, runtime Python portatile e sorgenti usati per preparare il pacchetto.

Il nome della cartella e storico: il contenuto e stato aggiornato alla versione indicata nel README e nel changelog.

## Community e sviluppo

`community/`

Contiene solo materiali di comunicazione e sviluppo:

- aggiornamenti sul lavoro in corso;
- roadmap voci;
- sample audio da ascoltare;
- richiesta feedback.

Questi file non sono necessari per installare la mod.

## GitHub Issue

`.github/ISSUE_TEMPLATE/`

Contiene il template per segnalare problemi audio o dare pareri sulle voci.
