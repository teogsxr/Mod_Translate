# Voice previews v0.4 work in progress

Sample audio della nuova passata ibrida ElevenLabs + modelli locali.
Questi file servono solo per ascoltare e lasciare feedback: non fanno parte della patch installabile.

## Ascolto

I link `Apri audio` portano alla pagina GitHub del singolo MP3. I link raw di GitHub forzano spesso il download, quindi qui non li uso come ascolto principale.

## Note di lettura

- I nomi dei file sono volutamente semplici: nome personaggio, con numero progressivo solo quando ci sono più sample dello stesso personaggio.
- Alcune voci sono ancora provvisorie e potrebbero cambiare nella prossima build.
- Marius e Kliff sono ancora in revisione qualitativa.
- Nei sample nuovi, la pronuncia guidata di Pailune punta a Peilùn, senza cambiare il testo a schermo.

## Feedback utili

- voce troppo finta o con accento straniero;
- personaggio che sembra avere età, tono o ruolo sbagliato;
- frase troncata o finale mangiata;
- emozione assente in scene di rabbia, paura, combattimento o dolore;
- pronunce sbagliate di nomi propri.
